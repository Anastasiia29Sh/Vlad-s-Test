Вопросы

Еще раз большое спасибо за помощь в HTML! Но у меня всё же осталось несколько вопросов.

  1. Что делает тег <div>?
  2. Зачем его используют?
  3. Чем он отличается от <span>?
  4. Что такое CSS и почему это часто упоминается с HTML?
  5. Зачем использовать CSS?
  6. Стоит ли мне учить CSS, если я хочу создавать веб странички?

Ответы Владу

  1. Тег div - это блочный элемент, он используется для выделения крупных элементов
  2. Тег div используют для деления веб-страницы на фрагменты. С его помощью можно объединить любой набор логически связанных элементов.
  3. Тег div отличается от тега span тем, что является блочным, то есть его содержимое будет начинаться с новой строки и занимать всю доступную ширину. Тег span - строчный, он применяется к внутренним элементам страницы, например, к фразам, которые находятся в пределах абзаца или фрагмента текста.
  4. CSS - это каскадная таблица стилей. Этот язык нужен для преобразования внешнего вида страницы.
  5. CSS позволяет прикреплять стиль. То есть HTML структурирует контент на странице, а CSS делает его привлекательным для читателей.
  6. Конечно стоит! HTML и CSS это базовые знания верстки сайта. Любой сайт создается с использованием этих двух языков.
