Есть 3 способа добавить CSS в HTML
  • Можно использовать глобальные стили CSS, добавив их в контейнер head HTML документа.
  • Можно использовать внешний .css файл, в нем прописать все необходимые вам стили. А в HTML документе добавить ссылку на данный файл с помощью тега link.
  • Можно использовать внутренний CSS, то есть прописывать стили к определенному элементу в HTML документе.
К первой строке применялись глобальные стили CSS. Они прописаны в контейнере head. (На желтом фоне зеленый текст).
Ко второй строке применялся внутренний CSS. Стили прописаны определенному параграфу p. (На красном фоне желтый текст).
К фону страницы применился внешний .css файл. (Цвет фона - светло серый). Также в данном файле прописаны стили для параграфов p (голубой фон и синий текст), но он не применился, так как в HTML документе параграфам были прописаны глобальные и внутренний стили, а они приоритетней внешнего css файла.
